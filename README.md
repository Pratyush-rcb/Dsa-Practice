# DSA Solutions

This repository contains solutions to Data Structures and Algorithms (DSA) problems I have solved on platforms like LeetCode, GeeksforGeeks, and Hackerrank.

## Topics Covered

- Arrays
  - Sliding Window
- Strings
  - Sliding Window
- LinkedLists
- Stacks
- Queues
- Trees
- Hashing / Maps
- Dynamic Programming (DP)
- Recursion
- Misc (Backtracking, Greedy, Bit Manipulation)

## How to Use

- Problems are organized by **topic**.
- Each file contains **comments at the top** specifying:
  - Problem name
  - Platform
  - Difficulty
- Add new problems to the relevant folder.
